# Agents

Repository for managing agent-related configurations, custom agents, and skills.

## Contents

- **Skills** - Custom skills and skill definitions for Claude Code agents
- **Custom Agents** - Agent configurations and implementations
- **Agent Configurations** - Setup files and hooks for agent behavior

## Purpose

This repository serves as a centralized location for:
- Defining and maintaining custom skills
- Configuring agent behavior and capabilities
- Managing agent-related settings and defaults
- Storing agent implementations and related tooling

## Getting Started

1. Review the structure to understand available skills and agents
2. Configure settings as needed for your specific use case
3. Integrate with Claude Code or other agent systems

For more information about skills and agents, refer to the documentation in respective directories.
